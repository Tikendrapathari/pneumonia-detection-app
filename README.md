# 🫁 Pneumonia Detection System using Deep Learning

## 📌 Project Overview

This project is a **Pneumonia Detection Web Application** that predicts whether a patient has pneumonia from chest X-ray images using Deep Learning.

The system allows users to upload an X-ray image and get a prediction instantly through a simple web interface.

---

## 🚀 Features

* Upload Chest X-ray images
* Automatic pneumonia prediction
* Simple and user-friendly web interface
* Machine learning based classification
* Fast prediction using trained model

## ⚙️ Installation

1️⃣ Clone the repository

```
git clone https://github.com/yourusername/pneumonia-detection-app.git
```

2️⃣ Go to project folder

```
cd pneumonia-detection-app
```

3️⃣ Install required libraries

```
pip install -r requirements.txt
```

4️⃣ Run the application

```
python app.py
```

5️⃣ Open browser

```
http://127.0.0.1:5000
```

## 🖥️ How It Works

1. User uploads a chest X-ray image
2. Image is processed using image processing techniques
3. Features are extracted
4. Machine learning model predicts **Pneumonia or Normal**
